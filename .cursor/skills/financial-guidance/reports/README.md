# Weekly reports

The financial-guidance skill **saves each weekly report here** (e.g. `2025-03-08-weekly.md`).

It also **reads past reports** to compare your current state to prior weeks and describe whether you’re trending in the right direction over time.

- One file per week; filename should include the date or week so they can be ordered.
- If you don’t want reports in version control, add `reports/*.md` (or similar) to `../.gitignore` and keep this README.
