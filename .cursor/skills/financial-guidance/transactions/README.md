# Transaction documents

**Put your bank transaction history / statements here** (CSV exports, PDFs, or other files your bank provides).

The financial-guidance skill reads everything in this folder to analyze your recent spending and compare it to your goals and plan in `goals.md`.

- Add new files each week when you run your weekly check-in.
- Files in this folder are **not** committed to git (see `../.gitignore`) so your data stays local.
